# AWS Basics

## Access tokens

| Prefix | Entity Type                  |
| ------ | ---------------------------- |
| ABIA   | AWS STS service bearer token |
| ACCA   | Context-specific credential  |
| AGPA   | Group                        |
| AIDA   | IAM user                     |
| AIPA   | Amazon EC2 instance profile  |
| AKIA   | Access key                   |
| ANPA   | Managed policy               |
| ANVA   | Version in a managed policy  |
| APKA   | Public key                   |
| AROA   | Role                         |
| ASCA   | Certificate                  |
| ASIA   | Temporary (AWS STS) keys     |
